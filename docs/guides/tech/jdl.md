JDownloader is a free download manager, but it comes with banner ads and other annoyances. In this guide you will learn how to remove them.

![](/ss/jd.png)

## Steps

:::steps
1. First download and install [**JDownloader 2**](https://jdownloader.org/jdownloader2).
2. Run JDownloader and open settings by pressing **Ctrl + P**.
3. Now go to **Advanced settings** (at the bottom of the left sidebar). Here we will modify the values of several keys to remove ads / annoyances. Click **Ok** in the confirmation pop-up.
4. Now search the mentioned and change the values:
    1. `banner` -> uncheck the value.
    2. `donate` -> set the value `Hidden (Automode)`
    3. `premium alert` -> uncheck all of the values
    4. `special deal` -> uncheck all of the values
:::
